## ğŸ“¦ Getting Started

### Prerequisites

- Node.js & npm
- Angular CLI
- .NET 8 SDK
- SQL Server

### Setup

1. **Clone the repository**
   ```sh
   git clone https://github.com/your-username/Virtual-Community-Support.git
   ```

2. **Frontend**
   ```sh
   cd Virtual-Community-Support/Frontend/AngularCode
   npm install
   ng serve
   ```

3. **Backend**
   ```sh
   cd Virtual-Community-Support/Backend/Mission/Mission
   dotnet restore
   dotnet build
   dotnet run
   ```
